# Quickstart

AxonHub Gateway Desk is a SaaS workspace for AI SDK gateway pricing.

## Fast Path

1. Open AxonHub Gateway Desk and choose a bounded pilot workflow.
2. Enter public-safe examples for AI SDK gateway pricing.
3. Review the generated queue, evidence, or approval state with the owner.
4. Use pricing and checkout links when the pilot needs a paid workspace.
5. Export the resulting report or evidence for the team record.
